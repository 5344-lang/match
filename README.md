# 라운드 테이블 매칭 시스템 💘

오프라인 모임의 매칭 시스템을 웹앱으로 구현한 프로젝트입니다. 
관리자는 직관적인 시각화 도구를 통해 전체 매칭 흐름을 파악하고, 참여자는 간편하게 서로의 성향을 확인하여 매칭을 신청할 수 있습니다.

## 주요 기능
- **회원 관리:** 개인 프로필 설정 (성향 스펙트럼 0~100 수치화)
- **프라이버시 보호:** 실제 커플 및 과거 매칭 이력은 관리자와 본인만 확인 가능
- **선택적 매칭:** 회차별 참여/휴식 여부 설정 및 다중 지망/기피 지목 기능
- **관리자 도구:** 합산 점수(100점) 기반 최적 조합 추천 및 파국 경고 팝업 기능 제공

## 기술 스택
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend / DB: Firebase Authentication, Firestore
- Hosting: GitHub Pages
